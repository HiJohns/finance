## 模块二：Go 风险哨兵 (The Sentinel)

**核心功能**：利用 Go 的并发优势进行 7x24 小时实时监控。

### 3. 实时止损与水位监控 (Price Monitor)

* **功能**：轮询行情 API。
* **逻辑**：
* **硬止损**：AMD 跌破设定的价格线（如当前回撤 15%）。
* **技术位**：价格跌破 200 日移动平均线 (MA200)。


* **输出**：即时桌面通知或邮件，防止“温水煮青蛙”。

### 4. 巨鲸异动监测 (Whale Tracker)

* **功能**：监控 SLV (白银) 和 GLD (黄金) 的**量价背离 (VPD)**。
* **逻辑**：如果成交量异常放大（Volume Spike）但价格滞涨甚至微跌，识别主力是否在“撤退”或“狼狈解套”（针对白银巨鲸分析）。

## 模块三：宏观与财务审计 (The Auditor)

**核心功能**：通过基本面数据证伪“AI 叙事”。

### 5. Mag 7 投入产出比监控 (AI-ROI Tracker)

* **数据源**：解析财务报表 (10-Q/10-K)。
* **因子计算**：
* **Capex Intensity**：资本支出 / 总收入。
* **FCF Yield**：自由现金流收益率。


* **功能**：监控 Mag 7 的烧钱效率。如果 **Capex 增速 > Revenue 增速**，触发组合中科技股 (AMD, IWY) 的预警减仓。

---

## 模块四：执行模块 (The Exec)

**核心功能**：解决“下单”层面的数学问题。

### 6. 整数规划执行 (Discrete Optimizer)

* **输入**：MWU 算出的浮点数权重（如 0.2543）。
* **算法**：使用 `cvxpy` 解决**整数背包问题 (Integer Knapsack)**。
* **目标**：在 A 股（100 股一手）和高价美股限制下，计算出最接近目标比例的真实购买股数。
